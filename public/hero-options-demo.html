Key Differences Between the Two Sets
Quatro Drupal Theme (Options 1–8)
  • Brand colour: #b70000 (Quatro red)
  • Font: Helvetica Neue / Arial Nova
  • Matches the live OAV Drupal website
  • Use when building the Drupal Twig template
  • Consistent with all other OAV pages
Microsite Theme (Options A–E)
  • Brand colour: #4eaac8 (STTT teal) + #8B1F2D (maroon accent)
  • Font: Gotham (licensed) / Helvetica Neue fallback
  • Matches the Figma-make React prototype
  • Use if the Spotlight page gets its own standalone design
  • Visually distinct from standard OAV pages — feels like a microsite
Recommendation: Use Quatro Theme options if the Spotlight page lives within the existing OAV Drupal site. Use Microsite Theme options if it's a standalone page or a React/headless build with its own design system.
